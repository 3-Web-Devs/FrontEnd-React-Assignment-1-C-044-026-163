Document-Type-Changer
Document-Type-Changer is a modern web application that allows users to easily convert, extract, and manipulate different document types. It provides a set of convenient tools for PDF, Word, Image, HTML, and scanned documents using OCR and AI integration.

Features / Services
PDF to Word Converter – Convert your PDFs into editable Word documents.
Image to PDF Converter – Convert images into PDF format.
HTML to PDF Converter – Convert web pages or HTML files into PDFs.
OCR PDF to Text – Extract text from scanned PDFs using Optical Character Recognition (OCR).
Tech Stack
Frontend: React.js
Styling: Tailwind CSS
Language: JavaScript (ES6+)
Backend / Server: Next.js
Database: MongoDB
AI Integration: OpenAI / HuggingFace API
Version Control: Git & GitHub
Features Overview
User-friendly interface for all conversions
Fast and reliable processing of documents
AI-powered OCR to extract text from scanned PDFs
Responsive design for desktop and mobile devices
Future-ready with AI APIs for smart enhancements
